C++简介

C++是一种被广泛使用的计算机程序设计语言。它是一种通用程序设计语言，支援多重编程模式，例如程序化程序设计、数据抽象)、面向对象程序设计、泛型程序设计和设计模式)等。

比雅尼·斯特劳斯特鲁普博士在贝尔实验室工作期间在20世纪80年代发明并实现了C++。起初，这种语言被称作“C with Classes”（“包含‘类)’的C语言”），作为C语言的增强版出现。随后，C++不断增加新特性。虚函数（virtual function）、运算符重载（operator overloading）、多继承（multiple inheritance）、标准模板库（standard template library, STL）、异常处理（exception）、运行时类型信息（Runtime type information）、命名空间（namespace）等概念逐渐纳入标准。1998年，国际标准组织（ISO）颁布了C++程序设计语言的第一个国际标准ISO/IEC 14882:1998，目前最新标准为ISO/IEC 14882:2017。根据《C++编程思想》（Thinking in C++）一书，C++与C的代码执行效率往往相差在±5%之间。

——维基百科（Wikipedia）

简答来讲，C++就是加强版的C语言，新增了很多新特性，极大地提高了编程效率，对于准备算法竞赛/考试的同学非常有学习的必要！

cin/cout输入输出

  • 头文件：<iostreanm>（输入输出流）
  • 使用方法：
    • cin >> n;等同于scanf("%d", &n); ，cin是向右的箭头，表示将内容输⼊到n中；cout << n; 等同于printf("%d", n);，cout是向左的箭头。
    • 无论n是double还是int或者是char类型，只需要写cin >> n; 和 cout << n; ，不需要像C语⾔中需要根据n的类型对应地写%lf、 %d、 %c。
    • endl等同于"\n" ，⼀般字符串推荐使用 “\n” ，变量推荐使用endl。
  • 效率：低于scanf/printf，如果题⽬对时间复杂度要求⽐较⾼，全都改成scanf和printf可以提⾼代码的输⼊输出效率 。

C++新特性

以下介绍一些在算法竞赛/考试常用的C++新特性 。对于准备算法竞赛/考试的同学，C++的面向对象特性不需要详细了解，这里不做介绍。

变量声明

  • C语⾔的变量声明⼀般都在函数的开头，但是C++在⾸次使⽤变量之前声明即可（当然也可以都放在函数的开头）。
  • C语言中在for循环的外⾯定义i变量，而C++⾥⾯可以在for循环内部定义且在for循环⾥⾯定义的局部变量，在循环外⾯失效，所以⼀个main函数⾥⾯可以定义好多次局部变量i，不⽤担⼼写的循环太多变量名i、 j、 k不够⽤。

bool型变量

C++引⼊了bool型变量，有false和true两个值。⽽且C++把所有⾮零值解释为true，零值解释为false，所以直接赋值⼀个数字给bool变量也是可以的。

结构体定义

定义好结构体之后，在使⽤这个结构体类型的时候， C语⾔需要写关键字struct，⽽C++⾥⾯可以省略不写。

引用“&”

下面的写法定义了一个引用，并将其初始化为引用某个变量。
类型名 & 引用名 = 某变量名;

某个变量的引用，等价于这个变量，相当于该变量的一个别名。

注意：要与C语⾔⾥⾯的取地址运算符&区分开来，两者无任何关系。

应用：在函数传入引用参数，那么对这个引⽤变量做的所有操作都是直接对传⼊的原变量进⾏的操作。

函数重载

函数重载：一个或多个函数，名字相同，然而参数个数或参数类型不相同。

  • 函数重载使得函数命名变得简单。
  • 编译器根据调用语句的中的实参的个数和类型判断应该调用哪个函数。

函数的缺省参数

C++中，定义函数的时候可以让最右边的连续若干个参数有缺省值，那么调用函数的时候，若相应位置不写参数，参数就是缺省值。

如果某个写好的函数要添加新的参数，而原先那些调用该函数的语句，未必需要使用新增的参数，那么为了避免对原先那些函数调用语句的修改，就可以使用缺省参数。

字符测试库<cctype>

cctype本质上是C语⾔标准函数库中的头⽂件 #include ，严格来讲，并不属于C++新特性的范畴，在⼀些字符串逻辑题的时候经常⽤到。

字符测试函数

常用函数：

  • isalpha（字⺟，包括⼤写、⼩写）
  • islower（⼩写字⺟）
  • isupper（⼤写字⺟）
  • isalnum（字⺟⼤写⼩写+数字）
  • isblank（space和\t）
  • isspace（space、 \t、 \r、 \n）

大小写转换

  • tolower将字符转为⼩写
  • toupper将字符转为⼤写
